Vous souvenez-vous de la mission "Univocalic Fecterel Golf" ? Pour cette nouvelle mission, vous utiliserez les même compétences, mais avec une petit modification : "a" ⇄ "e".

En mathématiques, la factorielle d'un nombre entier positif n, notée n!, est le produit de tous les nombres entiers naturels compris entre 1 et n.

Il s'agit d'écrire une fonction nommée "a_factaral" qui calcule la factorielle du paramètre, sans utiliser les lettres suivantes dans le code : "eiou".

Il y a une règle supplémentaire pour ce défi. C'est une mission code golf, et votre objectif est de faire le code le plus court possible. Plus votre code est court, plus vous gagnez de points. Votre score est calculé en fonction de la longueur de votre code (nombre de caractères utilisés): un code de 250 caractères ne rapportera aucun point ; pour chaque caractère au dessous de 250, vous gagnez un point. Par exemple, un code de 150 caractères rapporte 100 points.

Input: Un nombre entier naturel (int).

Output: La factorielle de ce nombre (int).

Application: Il s'agit des trucs et astuces en Python qui vous aident à faire un code concis. Ce n'est pas indispensable quand vous écrivez du code, mais cela aide à mieux comprendre le fonctionnement de Python.

Préconditions: 0 ≤ n < 100
